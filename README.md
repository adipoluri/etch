# Etch

A mobile-first, client-side PWA for tracing reference images. Load an image, view
it edge-to-edge fullscreen, convert it to traceable line art with on-device image
processing, then **lock** the screen so resting a hand or paper on the glass
can't move it.

100% client-side — no backend, no accounts, images never leave the device.

See [docs/SPEC.md](docs/SPEC.md) for the full v1 product spec.
